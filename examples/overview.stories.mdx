# Chart Gallery

The gallery holds examples of complete charts built using `chask` components.

## Milestone animations

Some examples are prepared with milestone animations. By default, these examples display the final state of the animation. This default makes it possible to grasp the overall form of all the charts while scrolling through the gallery.

To see the milestone animation, click the 'first-page' button in the chart toolbar, <span class="icon"><svg xmlns="http://www.w3.org/2000/svg" height="24" width="24"><path fill="#222222" d="M6 18V6h2v12Zm11 0-6-6 6-6 1.4 1.4-4.6 4.6 4.6 4.6Z" /></svg></span>. This will reset the chart state. Then, click the 'next' button, <span class="icon"><svg xmlns="http://www.w3.org/2000/svg" height="24" width="24"><path fill="#222222" d="M9.4 18 8 16.6l4.6-4.6L8 7.4 9.4 6l6 6Z" /></svg></span>, to step through the animation.

## Source code

The source code for all examples is available in [this directory](https://github.com/tkonopka/chask/tree/main/examples) of the [github repository](https://github.com/tkonopka/chask/).

In the gallery pages, each example is a separate storybook 'story' and carries a 'Show code' button in the lower-right corner. However, this button is not particularly useful in the gallery pages. This is because each example consists of data generation code and chart code, and is also wrapped in a controller that displays a toolbar. The 'Show code' button simply cannot reveal all these parts at once. Thus, to see the full source, please use the github repository instead.
