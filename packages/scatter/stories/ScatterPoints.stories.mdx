# ScatterPoints

<Meta title="Charts/Scatter/ScatterPoints" />
export const Template = args => <ScatterPoints {...args} />

`ScatterPoints` draws points on a scatter plot view.

<ArgsTable of={ScatterPoints} />

The examples on this page use a dataset with series with ids `linear` and `quadratic`.

## Series

Charts can display individual series by setting the prop `series`.

<Canvas>
    <Story
        name="a series"
        args={{
            series: 'linear',
            symbolStyle: { fill: '#dd3333' },
        }}
        component={ScatterPoints}
        decorators={[ChartScatterDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="another series"
        args={{
            series: 'quadratic',
            symbolStyle: { fill: '#dd3333' },
        }}
        component={ScatterPoints}
        decorators={[ChartScatterDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Multiple series

It is possible to use the `ScatterPoints` component multiple times and thus overlay multiple series.

<Canvas>
    <Story name="multiple series">
        <Chart size={[400, 300]} padding={[40, 40, 60, 60]} style={{ display: 'inline-block' }}>
            <Scatter
                data={data}
                x={'x'}
                y={'y'}
                r={5}
                scaleX={{
                    variant: 'linear',
                    min: 0,
                    max: 8,
                }}
                scaleY={{
                    variant: 'linear',
                    min: 0,
                    max: 64,
                }}
            >
                <Axis variant={'bottom'} label={'x (a. u.)'} />
                <Axis variant={'left'} label={'y (a. u.)'} ticks={5} />
                <ScatterPoints series={'linear'} symbolStyle={{ fill: '#dd3333' }} />
                <ScatterPoints series={'quadratic'} symbolStyle={{ fill: '#6666ee' }} />
            </Scatter>
        </Chart>
    </Story>
</Canvas>

## Symbols

Symbols can be adjusted with prop `symbol`.

<Canvas>
    <Story
        name="symbol"
        args={{
            series: 'quadratic',
            symbolStyle: { fill: '#ffdd66', stroke: '#222222', strokeWidth: '2px' },
            symbol: Triangle,
        }}
        component={ScatterPoints}
        decorators={[ChartScatterDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Note that the prop `symbol` requires a function. Some symbol functions are available in the `@chask/core` package: see `Circle`, `Diamond`, `Square`, `Triangle`, and `InvertedTriangle`.
