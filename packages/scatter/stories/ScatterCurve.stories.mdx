# ScatterCurve

<Meta title="Addons/Scatter/ScatterCurve" />
export const Template = args => <ScatterCurve {...args} />

`ScatterCurve` draws a curve based on a data series.

<ArgsTable of={ScatterCurve} />

Note that `ScatterCurve` uses all the points in a series. For noisy data, this may produce spiky lines. To display a best-fit line instead, see `ScatterRegression` instead.

The examples on this page use a dataset with series with ids `linear` and `quadratic`.

## Curves

The prop `curve` controls the type of interpolation between points.

## Multiple series

Use `ScatterCurve` multiple times to overlay multiple series. Each series can have a distinct curve setting and css style.

<Canvas>
    <Story name="multiple series">
        <Chart size={[400, 300]} padding={[40, 40, 60, 60]} style={{ display: 'inline-block' }}>
            <Scatter
                data={data}
                x={'x'}
                y={'y'}
                r={5}
                scaleX={{
                    variant: 'linear',
                    min: 0,
                    max: 8,
                }}
                scaleY={{
                    variant: 'linear',
                    min: 0,
                    max: 64,
                }}
            >
                <Axis variant={'bottom'} label={'x (a. u.)'} />
                <Axis variant={'left'} label={'y (a. u.)'} />
                <ScatterCurve
                    series={'linear'}
                    curve="Step"
                    style={{ stroke: '#dd3333', strokeWidth: 3 }}
                />
                <ScatterCurve series={'quadratic'} curve="Linear" style={{ stroke: '#6666ee' }} />
            </Scatter>
        </Chart>
    </Story>
</Canvas>
