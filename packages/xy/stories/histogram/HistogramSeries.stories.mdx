# HistogramSeries

<Meta title="Addons/XY/Histograms/HistogramSeries" />
export const Template = args => <HistogramSeries {...args} />

`HistogramSeries` is a convenience component that combines the capabilities of `HistogramArea`, `HistogramBars`, and `HistogramCurve`.

<ArgsTable of={HistogramSeries} />

## Layers

The prop `layers` controls the amount of information that is displayed about a series. Setting the prop to `['area', curve']` displays an area polygon and a curve.
