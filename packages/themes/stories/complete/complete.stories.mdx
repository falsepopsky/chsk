# Complete Themes

Complete themes specify a distinctive look-and-feel.

To apply one of the themes to a chart, import the corresponding object and pass it to the prop `theme` in a `Chart` component.

<Source
    language="javascript"
    code={`
import { classicTheme } from '@chask/themes'

<Chart {...props} theme={classicTheme}>
    ...
</Chart>
`} />

Complete themes can be augmented with theme pieces or with custom styles. See `mergeTheme`.

## `defaultTheme`

The default theme is applied automatically to all charts. The examples below show how some simple chart layout look with the default settings.

<Canvas>
    <Story name="default">

## `inverseGridTheme`

The inverse grid theme is characterized by white grid lines.

<Canvas>
    <Story name="inverse grid">
        <ThemeController
            chart={ThemeScatterChart}
            chartId={'chartInverseGrid'}
            themes={{ inverseGridTheme, defaultTheme }}
        />
    </Story>
</Canvas>

## `boxedTheme`

The boxed theme is characterized by axes that are offset slightly from the edges of the view. `Surface` components create a bounding box around a view.

    <Story name="boxed">

            themes={{ boxedTheme, defaultTheme }}
        />
    </Story>
</Canvas>
