# Quantiles

<Meta title="Addons/Band/Quantiles" />
export const Template = args => <Quantiles {...args} />

`Quantiles` draws boxes and whiskers to summarize distributions of numeric values.

<ArgsTable of={Quantiles} />

Note that the `Quantiles` component is intended to be used within a `Quantile` component. The `Quantiles` component is the one that draws the bars and whiskers, but much of the layout is determined by `Quantile`.

## Data subsets

By default, `Quantiles` draws the entire dataset. To draw subsets of bars, set the props `ids` and `keys`.

<Canvas>
    <Story
        name="selected ids"
        args={{

            whiskerStyle: { strokeWidth: 2 },

## Whisker cap width

The size of the whisker 'cap' can be adjusted with prop `whiskerCapWidth`.

<Canvas>
    <Story
        name="whisker cap"
        args={{

            boxStyle: { strokeWidth: 0.5, stroke: '#444444' },
            whiskerCapWidth: 0.5,
        }}
        component={Quantiles}
        decorators={[ChartQuantileHorizontalDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
