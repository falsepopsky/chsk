# Bars

<Meta title="Addons/Band/Bars" />
export const Template = args => <Bars {...args} />

`Bars` draws a set of rectangles to display a bar chart.

<ArgsTable of={Bars} />

Note that the `Bars` component is intended to be used within a `Bar` component. The `Bars` component is the one that draws the bars, but actually much of the layout is determined by the parent component `Bar`. The examples on this page demonstrate the features in both `Bar` and `Bars`.

## Orientation and Stacking

Props `horizontal` and `stacking` determine the direction of the bars and whether they appear one-after-the-other or side-by-side.
