# Bars

<Meta title="Addons/Band/Bars" />
export const Template = args => <Bars {...args} />

`Bars` draws a set of rectangles to display a bar chart.

<ArgsTable of={Bars} />

Note that the `Bars` component is intended to be used within a `Bar` component. The `Bars` component is the one that draws the bars, but actually much of the layout is determined by `Bar`.

## Data subsets

By default, `Bars` draws the entire dataset. To draw subsets of bars, set the props `ids` and `keys`.
