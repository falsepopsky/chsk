# Bars

<Meta title="Addons/Band/Bars" />
export const Template = args => <Bars {...args} />

`Bars` draws a set of rectangles to display a bar chart.

`Bars` is intended to be used within a `Bar` component. The `Bars` component is the one that draws visual elements, but actually much of the layout is determined by `Bar`.

<ArgsTable of={Bars} />

## Data subsets

By default, `Bars` draws the entire dataset. To draw subsets of bars, set the props `ids` and `keys`.

<Canvas>
    <Story
        name="selected ids"
        args={{ ids: ['alpha'], style: { strokeWidth: 1, stroke: '#222222' } }}
        component={Bars}
        decorators={[ChartBarH0S1Decorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="selected keys"
        args={{ keys: ['x', 'z'], style: { strokeWidth: 1, stroke: '#222222' } }}
        component={Bars}
        decorators={[ChartBarH0S1Decorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Mouse events

To make bars respond to mouse events, set props starting with the prefix `on`, for example `onClick`.

<Canvas>
    <Story
        name="mouse click"
        args={{ onClick: onBarsClick }}
        component={Bars}
        decorators={[ChartBarH0S1Decorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

(Check the browser console with `Ctrl Shift J` for output.)

## Tooltip

To make a tooltip appear upon hovering the mouse on a bar, there are two steps needed.

First, prop `dataComponent` should be set with a functional component that forwards data to the tooltip context.
`TooltipDataComponent` from `@chsk/core` is a suitable choice, or it can act as a template for customization.

Second, the chart code should be extended with a `Tooltip` component.

<Canvas>
    <Story name="tooltip">
        <Chart size={[400, 300]} padding={[40, 40, 60, 60]} style={{ display: 'inline-block' }}>
            <Bar {...commonBarProps} horizontal={false} variant={'stacked'}>
                <GridLines variant={'y'} />
                <Axis variant={'bottom'} />
                <Axis variant={'left'} label={'Values (a.u.)'} />
                <Bars dataComponent={TooltipDataComponent} />
                <Tooltip position={[0, -10]} rx={2} ry={2} anchor={[0.5, 1]} />
            </Bar>
        </Chart>
    </Story>
</Canvas>
