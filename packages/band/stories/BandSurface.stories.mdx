# BandSurface

<Meta title="Addons/Band/BandSurface" />
export const Template = args => <BandSurface {...args} />

`BandSurface` creates rectangles that highlight bands.

<ArgsTable of={BandSurface} />

## Selective highlights

Use prop `ids` to specify the bands to highlight.

<Canvas>
    <Story name="static">
        <Chart
            id={'bh2'}
            size={[400, 240]}
            padding={[60, 60, 60, 60]}
            style={{ display: 'inline-block' }}
        >
            <Bar {...commonBarProps} horizontal={true} stacked={false}>
                <BandSurface ids={['alpha']} style={{ fill: '#dddddd' }} />
                <Bars />
                <Axis variant={'top'} label={'Values (a.u.)'} />
                <Axis variant={'left'} />
            </Bar>
        </Chart>
    </Story>
</Canvas>

## Interactivity

Flag `interactive` activates interactivity. Hover the mouse on the examples below to make the surfaces appear.

<Canvas>
    <Story name="highlights 1">
        <Chart
            id={'bh1'}
            size={[400, 240]}
            padding={[60, 60, 60, 60]}
            style={{ display: 'inline-block' }}
        >
            <Bar {...commonBarProps} horizontal={true} stacked={false}>
                <BandSurface interactive={true} style={{ fill: '#eeeeee' }} />
                <Bars />
                <Axis variant={'top'} label={'Values (a.u.)'} />
                <Axis variant={'left'} />
            </Bar>
        </Chart>
    </Story>
    <Story name="highlights 2">
        <Chart
            id={'bh2'}
            size={[400, 240]}
            padding={[60, 60, 60, 60]}
            style={{ display: 'inline-block' }}
        >
            <Bar {...commonBarProps} horizontal={true} stacked={false}>
                <BandSurface interactive={true} style={{ fill: '#eeeeee' }} />
                <Bars style={{ 'pointer-events': 'none' }} />
                <Axis variant={'top'} label={'Values (a.u.)'} />
                <Axis variant={'left'} />
            </Bar>
        </Chart>
    </Story>
</Canvas>

When a band contains content, for example bars in a bar chart, then hovering on that content may not activate the highlight (see first example above). To make the highlight visible, the style on that content must include `pointer-events: 'none'`. This can be set inline (see second example above), or via themes.
