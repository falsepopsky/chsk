# UpSet

<Meta title="Addons/Matrix/UpSet" />

export const Template = args => <UpSet {...args} />

`UpSet` sets up a view for an upset chart.

<ArgsTable of={UpSet} />

## Data

Data should be prepared as an array of objects. All objects should contain a string identifier `'id'` and an array of data elements `'data'`. The data elements will be treated as sets, and the UpSet chart will summarize overlaps between the sets.

The pseudocode below shows one data item with an identifier and a data array consisting of four strings.

<Source
    language="javascript"
    code={`
[
  {
    "id": "alpha",
    "data": ['a', 'b', 'c', 'd'],
  },
  ...
]
`}
/>

## Orientation

`UpSet` charts can be oriented horizontally (default) or vertically. The prop `horizontal` toggles the orientation of the axis that summarizes set combinations.

<Canvas>
    <Story
        name="default"
        args={{
            ...commonUpSetProps,
            children: (
                <>
                    <Axis variant="left" />
                    <UpSetGrid symbolStyle={{ fill: '#ccc' }} />
                </>
            ),
        }}
        component={UpSet}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="vertical"
        args={{
            ...commonUpSetProps,
            horizontal: false,
            children: (
                <>
                    <Axis variant="top" />
                    <UpSetGrid symbolStyle={{ fill: '#ccc' }} />
                </>
            ),
        }}
        component={UpSet}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Symbol size

Grid symbols are sized automatically to fill the space available in each row / column cell. To modulate the apparent size of the symbol, set the padding for the x-axis and y-axis band scales.

<Canvas>
    <Story
        name="large padding"
        args={{
            ...commonUpSetProps,
            scaleIndex: { variant: 'band', padding: 0.5 },
            scaleMembership: { variant: 'band', padding: 0.5 },
            children: (
                <>
                    <Axis variant="left" />
                    <UpSetGrid symbolStyle={{ fill: '#ccc' }} />
                </>
            ),
        }}
        component={UpSet}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## `HeatMap` components

Some components designed for heat maps can also be used within `UpSet` views.
In particular, `HeatMapSurface` can provide outlines or background colors for the UpSet grid.
