# HeatMapCircle

<Meta title="Charts/Matrix/HeatMapCircle" />
export const Template = args => <HeatMapCells {...args} />

`HeatMapCircle` draws a single circle. This component can be used together with `HeatMapCells` to draw a heatmap with custom cells.

<ArgsTable of={HeatMapCircle} />

Note that `HeatMapCircle` has the same API as `Rectangle`, and uses its props to display a circle instead of a rectangle.
