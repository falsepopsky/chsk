# VerticalGoldenRectangle

export const Template = args => <VerticalGoldenRectangle {...args} />

`VerticalGoldenRectangle` displays a rectangular symbol with height / width proportion set to the golden ratio (1.618).

Geometrically, golden rectangles are just rectangles and can be created using the versatile `Rectangle` component.
However, `VerticalGoldenRectangle` is relevant in that it has the same API as symbols such as `Circle` and `Square`.
The component can be used, for example, in legends and tooltips.
