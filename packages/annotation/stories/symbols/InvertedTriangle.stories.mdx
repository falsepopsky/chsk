# InvertedTriangle

export const Template = args => <InvertedTriangle {...args} />

`InvertedTriangle` displays a triangle shape.

Note that `InvertedTriangle` is similar to `Triangle`, except the shape is flipped upside down.

<ArgsTable of={InvertedTriangle} />
