# `useMilestone`

The `useMilestone` hook retrieves the current chart 'milestone' string.

<Source
    language="javascript"
    code={`
import { useMilestone } from '@chask/core'

// in a JSX component:
const milestone = useMilestone()
`} />

Recall milestone strings are used to trigger animations with `MilestoneMotion`.
