# Rectangle

<Meta title="Core/Components/Rectangle" />
export const Template = args => <Rectangle {...args} />

`Rectangle` displays a rectangle shape.

<ArgsTable of={Rectangle} />

## Centering

By default, props `x` and `y` specify the coordinates for the top-left corner of the rectangle.
To use those coordinates as the center of the rectangle instead, set the prop `center` to `true`.

<Canvas>
    <Story
        name="centered"
        args={{
            style: { fill: '#dd9999' },
            x: 0,
            y: 0,
            width: 60,
            height: 30,
            center: true,
        }}
        component={Rectangle}
        decorators={[DivChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Rounded corners

Rectangle corners can be rounded by setting props `rx` and `ry`.

<Canvas>
    <Story
        name="rounded corners"
        args={{
            style: { fill: '#dd9999', strokeWidth: '1px', stroke: '#aa7777' },
            x: 30,
            y: 20,
            width: 240,
            height: 90,
            rx: 30,
            ry: 20,
        }}
        component={Rectangle}
        decorators={[DivChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
