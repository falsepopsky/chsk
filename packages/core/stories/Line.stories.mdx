import { ArgsTable, Canvas, Meta, Story } from '@storybook/addon-docs'
import { Chart, Line } from '../src'
import { ChartDecorator } from './decorators'

# Line

<Meta title="Core/Components/Line" />
export const Template = args => <Line {...args} />

`Line` draws a single line on a chart.

<ArgsTable of={Line} />

## Variants

There are a number of predefined variants, which can be activated via the prop `variant`.

<Canvas>
    <Story
        name="tick"
        args={{
            variant: 'tick',
            x1: 140,
            y1: 80,
            x2: 140,
            y2: 86,
        }}
        component={Line}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="grid"
        args={{
            variant: 'grid',
            x1: 0,
            y1: 40,
            x2: 320,
            y2: 40,
        }}
        component={Line}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

The examples above show variants `tick` and `grid`. Other variants include `default` and `axis`.
(The default theme does not display a line for axes, hence it would not be useful to display an example.)

## Styles

It is possible to adjust individual lines using css.

<Canvas>
    <Story
        name="color"
        args={{
            variant: 'default',
            x1: 0,
            y1: 80,
            x2: 280,
            y2: 0,
            style: {
                stroke: '#dd4444',
            },
        }}
        component={Line}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="dasharray"
        args={{
            variant: 'default',
            x1: 40,
            y1: 0,
            x2: 320,
            y2: 80,
            style: {
                stroke: '#222222',
                strokeDasharray: '15,15',
                strokeLinecap: 'round',
                strokeWidth: 5,
            },
        }}
        component={Line}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
