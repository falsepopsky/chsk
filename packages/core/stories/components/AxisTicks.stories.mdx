# AxisTicks

<Meta title="Core/Components/AxisTicks" />
export const Template = args => <AxisTicks {...args} />

`AxisTicks` draws a set of tick line and labels.

`AxisTicks` should be used within an `Axis` component.

<ArgsTable of={AxisTicks} />

## Variants

Predefined variants draw ticks on the top, right, bottom, or left axis. For brevity, the example below shows ticks on the bottom axis.

<Canvas>
    <Story
        name="default"
        args={{
            variant: 'bottom',
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Tick marks

The position of ticks can be specified as an integer (to suggest a number of tick marks) or as an array (to specify all the tick positions).

<Canvas>
    <Story
        name="number"
        args={{
            variant: 'bottom',
            ticks: 10,
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="positions"
        args={{
            variant: 'bottom',
            ticks: [0, 20, 50, 90, 100],
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Note that suggestions for the number of tick marks are approximate and the actual number of ticks may differ; the actual positions are computed by [d3-scale](https://github.com/d3/d3-scale).

## Tick size and label position

Props `tickSize` and `labelOffset` control the length of the tick lines and the space between the axis and the text labels, respectively. These two props are often used together.

<Canvas>
    <Story
        name="size and padding"
        args={{
            variant: 'bottom',
            tickSize: 15,
            labelOffset: 30,
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Rotation

Tick labels can be rotated using the prop `labelRotate`. Note that adjusting the angle will likely prompt adjustment to css properties like `textAnchor` and `dominantBaseline`.

<Canvas>
    <Story
        name="rotation"
        args={{
            variant: 'bottom',
            labelOffset: 10,
            labelRotate: -90,
            labelStyle: {
                textAnchor: 'end',
                dominantBaseline: 'middle',
            },
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Label format

Labels can be formatted with a custom function through the prop `labelFormat`.

<Canvas>
    <Story
        name="format"
        args={{
            variant: 'bottom',
            labelOffset: 10,
            labelFormat: v => v + '%',
        }}
        component={AxisTicks}
        decorators={[ChartBottomAxisDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
