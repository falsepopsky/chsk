# GoldenRect

<Meta title="Core/Components/GoldenRect" />
export const Template = args => <GoldenRect {...args} />

`GoldenRect` displays a rectangular symbol with width-height proportions set to the golden ratio (phi = 1.618).

<ArgsTable of={GoldenRect} />

Geometrically, golden rectangles are just rectangles and can be created using the versatile `Rectangle` component. However, `GoldenRect` is relevant in that its API is equivalent to `Circle` and other symbols like `Triangle` or `Diamond`. This component can be used, for example, in legends.
