# Segment

<Meta title="Core/Components/Shapes/Segment" />
export const Template = args => <Segment {...args} />

`Segment` displays a short horizontal line segment. It is intended as a symbol for legends.

<ArgsTable of={Segment} />

## Legends

`Segment` accepts the same props as symbols like `Circle`, `Square`, and others. The component can thus be used as a symbol in a legend. It can also be styled using css.
