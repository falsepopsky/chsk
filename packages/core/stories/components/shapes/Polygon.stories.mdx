# Polygon

<Meta title="Core/Components/Shapes/Polygon" />
export const Template = args => <Polygon {...args} />

`Polygon` displays a closed shape composed of straight-line segments.

<ArgsTable of={Polygon} />

## Style

export const points = [
    [50, 120],
    [140, 50],
    [270, 90],
    [200, 190],
    [100, 100],
]

Polygons are defined by a series of points. The resulting shape can be styled with css.

<Canvas>
