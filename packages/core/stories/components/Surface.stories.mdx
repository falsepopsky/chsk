import { ArgsTable, Canvas, Meta, Story } from '@storybook/addon-docs'
import { Chart, Surface } from '../../src'
import { ChartDecorator } from './decorators'

# Surface

<Meta title="Core/Components/Surface" />
export const Template = args => <Surface {...args} />

`Surface` provides a rectangle intended as a background.

<ArgsTable of={Surface} />

## Variants

There are two preset variants: `inner` and `outer`.

<Canvas>
    <Story
        name="inner surface"
        args={{
            variant: 'inner',
            style: { fill: '#eeeeee' },
        }}
        component={Surface}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="outer surface"
        args={{
            variant: 'outer',
            style: { fill: '#eeeeee' },
        }}
        component={Surface}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

It is also possible to use them together.

<Canvas>
    <Story name="multiple surfaces">
        <Chart size={[400, 300]} padding={[40, 40, 40, 40]}>
            <Surface variant={'outer'} style={{ fill: '#eeeeee' }} />
            <Surface variant={'inner'} style={{ fill: '#cccccc' }} />
        </Chart>
    </Story>
</Canvas>

## Expansion

The predefined variants can be adjusted via the prop `expansion`, which stretches a surface in four directions.

<Canvas>
    <Story
        name="expanded"
        args={{
            variant: 'inner',
            style: { fill: '#eeeeee' },
            expansion: [40, 0, 0, 0],
        }}
        component={Surface}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
