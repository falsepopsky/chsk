import { ArgsTable, Canvas, Meta, Story } from '@storybook/addon-docs'
import { Chart, Surface, Typography } from '../../src'

# Typography

<Meta title="Core/Components/Typography" />
export const Template = args => <Typography {...args} />

export const ChartDecorator = Story => (
    <Chart
        size={[400, 200]}
        padding={[60, 60, 60, 60]}
        style={{ margin: '0.5em', border: 'solid 1px #aa3333', display: 'inline-block' }}
    >
        <Surface variant={'inner'} />
        {Story()}
    </Chart>
)

`Typography` places a text element on a chart.

<ArgsTable of={Typography} />

## Variants

There are a number of predefined variants, which can be activated via the prop `variant`.

<Canvas>
    <Story
        name="title"
        args={{
            variant: 'title',
            position: [0, -30],
            children: 'Primary heading',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="subtitle"
        args={{
            variant: 'subtitle',
            position: [0, -10],
            children: 'Secondary heading',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="axisLabel"
        args={{
            variant: 'axisLabel',
            position: [140, 115],
            children: 'axis label',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="tickLabel"
        args={{
            variant: 'tickLabel',
            position: [0, 85],
            children: 'tick label',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

In the examples above, the prop `variant` determines the visual style for the text.
Positions are set manually to suggest how the visual styles might be used within a chart.

## Styles

Components can be adjusted individually using css styles and svg transforms.

<Canvas>
    <Story
        name="color"
        args={{
            variant: 'title',
            position: [0, -30],
            children: 'Title in color',
            style: {
                fill: '#dd4444',
                alignmentBaseline: 'middle',
            },
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="rotation"
        args={{
            variant: 'default',
            position: [0, 0],
            children: 'Text with custom style',
            style: {
                fill: '#222222',
                fontFamily: 'serif',
                letterSpacing: 2,
                fontSize: '20px',
                textAnchor: 'middle',
                alignmentBaseline: 'middle',
            },
            transform: 'translate(140, 40) rotate(-10)',
        }}
        component={Typography}
        decorators={[ChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Note that using the prop `transform` automatically disables animations. In particular, text rotations are not animated.
