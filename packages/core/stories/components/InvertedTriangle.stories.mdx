# InvertedTriangle

<Meta title="Core/Components/InvertedTriangle" />
export const Template = args => <InvertedTriangle {...args} />

`InvertedTriangle` displays a triangle shape.

<ArgsTable of={InvertedTriangle} />

Note that `InvertedTriangle` is very similar to `Triangle`, except the shape appear upside-down.

## Styles

Inverted triangles can be styled using css.
