# LinearGradient

<Meta title="Core/Components/LinearGradient" />
export const Template = args => <LinearGradient {...args} />

`LinearGradient` is a component that defines a color gradient. Definitions are invisible themselves, but they can be used to decorate other components.

`LinearGradient` objects should be contained within an svg `def` tag.

<ArgsTable of={LinearGradient} />

## Two-color gradients

The simplest gradients define just two colors with prop `stops`.

<Canvas>
    <Story name="horizontal" decorators={[DivDecorator]}>
        <Chart size={[400, 300]}>
            <defs>
                <LinearGradient
                    id="gradient-horizontal"
                    start={[0, 0]}
                    end={[1, 0]}
                    stops={['#dddddd', '#ffdddd']}
                />
            </defs>
            <Surface variant={'inner'} style={{ fill: 'url(#gradient-horizontal)' }} />
        </Chart>
    </Story>
    <Story name="vertical" decorators={[DivDecorator]}>

## Custom colors, offsets, directions

Props `stops` and `offsets` can be used to define complex gradients with multiple colors and spacings between colors.
Props `start` and `end` can be used to set the direction for the color progression.

<Canvas>
    <Story name="custom" decorators={[DivDecorator]}>
        <Chart size={[400, 300]}>
            <defs>
                <LinearGradient
                    id="gradient-custom"
                    start={[0, 1]}
                    end={[1, 0]}
                    stops={['#dddddd', '#44ffdd', '#ffff44']}
                    offsets={[0, 0.3, 0.7]}
                />
            </defs>
            <Surface variant={'inner'} style={{ fill: 'url(#gradient-custom)' }} />
        </Chart>
    </Story>
</Canvas>
