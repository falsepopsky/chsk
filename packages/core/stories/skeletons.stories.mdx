# Chart skeletons

Core components can be combined to produce distinctive chart skeletons.

(Chart skeletons are charts that only display chart infrastructure, for example axes and labels, but do not yet display data.)

## Simple

<Canvas>
    <Story name="default">
        <Chart {...commonProps}>
            <View {...commonViewProps}>

                <Axis variant={'left'} label={'y-axis (a.u.)'} />

            </View>
        </Chart>

</Canvas>
