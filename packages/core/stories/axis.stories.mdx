# Axis

`Axis` draws an axis on a chart.

## Variants

Predefined variants places an axis on the top, right, bottom, and left sides of a chart.

## Examples

Tick labels can be rotated through css styles. Note that adjusting the text angle will likely prompt adjustment to other css properties like `textAnchor` and `dominantBaseline`.

<Canvas>
    <Story
        name="left rotated"
        args={{
            variant: 'left',
            label: 'axis label',
            ticks: 5,
            tickLabelStyle: {
                transform: 'rotate(-90deg)',
                textAnchor: 'middle',
                dominantBaseline: 'auto',
            },
        }}
        component={Axis}
        decorators={[DivChartDecoratorLargeMargin]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="bottom rotated"
        args={{
            variant: 'bottom',
            label: 'axis label',
            ticks: 5,
            tickLabelStyle: {
                textAnchor: 'end',
                transform: 'rotate(-90deg)',
                dominantBaseline: 'middle',
            },
        }}
        component={Axis}
        decorators={[DivChartDecoratorLargeMargin]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
