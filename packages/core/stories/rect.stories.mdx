# Rect

<Meta title="Core/Components/Rect" />
export const Template = args => <Rect {...args} />

`Rect` displays a rectangle shape.

<ArgsTable of={Rect} />

## Examples

Rectangles can be tuned using css.

<Canvas>
    <Story
        name="fill color"
        args={{
            style: { fill: '#dd9999' },
            x: 0,
            y: 0,
            width: 100,
            height: 50,
        }}
        component={Rect}
        decorators={[DivChartDecorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="stroke color"
        args={{
            style: { fill: '#dd9999', stroke: '#662222', strokeWidth: '2px' },
            x: 40,
            y: 40,
            width: 60,
            height: 140,
        }}
        component={Rect}
        decorators={[DivChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

By default, props `x` and `y` specify the coordinates for the top-left corner of the rectangle.
To use those coordinates as the center of the rectangle instead, set the prop `center` to `true`.

<Canvas>
    <Story
        name="centered"
        args={{
            style: { fill: '#dd9999' },
            x: 0,
            y: 0,
            width: 40,
            height: 40,
            center: true,
        }}
        component={Rect}
        decorators={[DivChartDecorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
