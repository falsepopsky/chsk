# Square

<Meta title="Core/Components/Square" />
export const Template = args => <Square {...args} />

`Square` displays a square shape.

<ArgsTable of={Square} />

Geometrically, squares are a special types of rectangles. Similar in chask, square shapes can be produced using the `Rectangle` component. Indeed, `Rectangle` is a more versatile component with more features than `Square`. However, `Square` is relevant in that its position is always set via the shape's center.
