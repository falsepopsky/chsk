# VennSetLabels

<Meta title="Addons/Set/VennSetLabels" />

export const Template = args => <VennSetLabels {...args} />

`VennSetLabels` draws labels on a Venn chart.

<ArgsTable of={VennSetLabels} />

## Data subsets

By default, `VennSetLabels` draws labels for all the sets in the dataset. To draw subsets, specify the desired sets via prop `ids`.

<Canvas>
    <Story name="default" component={VennSetLabels} decorators={[ChartVennSets2Decorator]}>
        {Template.bind({})}
    </Story>
    <Story
        name="one set"
        args={{ ids: ['alpha'] }}
        component={VennSetLabels}
        decorators={[ChartVennSets2Decorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Label position

By default, labels appear at the centers of the sets. The positions can be adjusted with props `rs`, `angles`, and `translate`. The first two props displace the labels in a radial and angular direction. The radial distance is relative to the size of the sets, so labels with radii set to unity are placed on / near the circumference. Translations are absolute shifts.

<Canvas>
    <Story
        name="labels under sets"
        args={{ rs: [1, 1], angles: [Math.PI, Math.PI], translate: [0, 16] }}
        component={VennSetLabels}
        decorators={[ChartVennSets2Decorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

For labels at custom angles, it may be desirable to individual sets with separate `VennSetLabels` components, each with its own custom settings.

<Canvas>
    <Story name="labels at custom angles">
        <Chart size={[400, 300]} padding={[80, 60, 60, 80]} style={{ display: 'inline-block' }}>
            <Venn {...commonVenn2Props}>
                <VennSets style={{ fillOpacity: 0.7 }} />
                <VennSetLabels
                    ids={['alpha']}
                    rs={[1]}
                    angles={[-Math.PI / 3]}
                    translate={[0, -28]}
                />
                <VennSetLabels
                    ids={['beta']}
                    rs={[1]}
                    angles={[+Math.PI / 3]}
                    translate={[0, -28]}
                />
            </Venn>
        </Chart>
    </Story>
</Canvas>

## Label format

By default, labels display the set id. The content can be adjusted via prop `format`.

<Canvas>
    <Story
        name="custom format"
        args={{
            format: x => x.substr(0, 1).toLocaleUpperCase(),
        }}
        component={VennSetLabels}
        decorators={[ChartVennSets2Decorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
