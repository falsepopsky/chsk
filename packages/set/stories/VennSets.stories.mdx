# VennSets

<Meta title="Addons/Set/VennSets" />

export const Template = args => <VennSets {...args} />

`VennSets` draws a set of symbols to represent sets on a Venn chart.

<ArgsTable of={VennSets} />

## Data subsets

By default, `VennSets` draws all sets in the dataset. To draw subsets, set the props `ids`.

<Canvas>
    <Story name="default" component={VennSets} decorators={[ChartVenn2Decorator]}>
        {Template.bind({})}
    </Story>
    <Story
        name="one set"
        args={{ ids: ['alpha'] }}
        component={VennSets}
        decorators={[ChartVenn2Decorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

## Styling

Sets can be styled using css.

<Canvas>
    <Story
        name="styled - two sets"
        args={{ style: { strokeWidth: 1, stroke: '#222', fillOpacity: 0.7 } }}
        component={VennSets}
        decorators={[ChartVenn2Decorator]}
    >
        {Template.bind({})}
    </Story>
    <Story
        name="styled - three sets"
        args={{ style: { strokeWidth: 1, stroke: '#222', fillOpacity: 0.7 } }}
        component={VennSets}
        decorators={[ChartVenn3Decorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>

Note that the colors of the sets are determined by the color scale, which should be set in the parent `Venn` view.

## Set symbol

The usual way to display sets on a Venn diagram is using circles, and this is the default behavior for `VennSets`. However, it is possible to use a different shape by setting prop `component`.

<Canvas>
    <Story
        name="squares"
        args={{
            component: Square,
            style: { fillOpacity: 0.7, strokeWidth: 1, stroke: '#222' },
        }}
        component={VennSets}
        decorators={[ChartVenn2Decorator]}
    >
        {Template.bind({})}
    </Story>
</Canvas>
