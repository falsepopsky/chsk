# ScatterSeries

<Meta title="Charts/Scatter/ScatterSeries" />

`ScatterSeries` is a convenience component that combines the capabilities of `ScatterArea`, `ScatterCurve`, `ScatterInterval`, and `ScatterPoints`.

<ArgsTable of={ScatterSeries} />

export const dataWithInterval = [
    generateScatterSeriesWithInterval(
        'A',
        Array(16)
            .fill(0)
            .map((v, i) => i),
        x => 1 + 0.3 * x + Math.random() * 1.5,
        [-0.8, 1.0]
    ),
    generateScatterSeriesWithInterval(
        'B',
        Array(16)
            .fill(0)
            .map((v, i) => i),
        x => 8 - 0.3 * x + Math.random() * 1.5,
        [-0.8, 1.0]
    ),
]

## Layers

The prop `layers` controls the amount of information that is displayed about a series. Setting the prop to an array `['curve', 'points']`, for example, can display a curve and data points. An array with an additional `'area'` would add an area polygon.
